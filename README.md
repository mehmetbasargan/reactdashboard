# React + TypeScript + Vite

## Başlarken

ilk önce bağımlılıkları yükleyin:

```bash
npm install
# or
yarn
```

Sonra geliştirme sunucusunu çalıştırın:

```bash

npm run dev
# or
yarn dev

```

Sonucu görmek için tarayıcınızla [http://localhost:5173](http://localhost:5173) dosyasını açın.
